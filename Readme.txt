tested on python 3.10.5
to install python packages:
(optional) create virtual environment: "python -m venv venv" or "python3 -m venv venv", then activate through "source venv/bin/activate"
install python packages in requirements.txt: "pip install -r requirements.txt"
to run programs:
python matplotlib_example_plotting_from_csv_onto_figure.py
python matplotlib_example_plotting_from_xlsx_onto_figure.py
python bokeh_example_plotting_from_csv_onto_html_page.py
python bokeh_example_plotting_from_xlsx_onto_html_page.py
